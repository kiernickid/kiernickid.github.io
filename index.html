About the visualization

Hybrid Structure - I chose to build an interactive slideshow using the reveal.js presentation framework. By using the right and left arrows on the keyboard or clicking the button, the reader is able to move around between various scenes while keeping track of progress by looking at the bar at the very bottom. The user is also able to interact with the d3 charts as described later on.

Scenes - The slides in my slideshow correspond to scenes, and I used a consistent beige background template styled by css. To tell my story, I ordered my charts with the bubble chart first to give an overview of the top cryptocurrencies before diving deeper with the line graphs to portray exact price patterns of some of the most prominent cryptos.

Annotations - In my line graph I used two annotations to highlight 2 of the most significant events in the Bitcoin timeline. The first event was the peak that Bitcoin enjoyed towards the end of 2017 when the price of a single Bitcoin neared $20K! The other event was when the Bitcoin fork occurred in August 2017, giving rise to Bitcoin Cash, and we can clearly see the (red) BCH line come up at this time. The annotations are cleared when the user leaves this particular scene.

About the visualization (cont.)

Parameters - There are several parameters that my visualization utilizes such as the current slide state in the context of the overall presentation (slide number), whether or not a particular line should be displayed based on whether or not the symbol has been clicked, and whether or not a mouseover has occurred for the bubble chart.

Triggers - There are a few triggers that my narrative visualization utilizes. As mentioned above, the pressing right/left arrow key or clicking the corresponding button will cause the slide to advance or go back. For the bubble chart, mousing over each bubble will display a textbox with the name and actual marketcap value. Finally, clicking the crypto symbol for my line graph causes the corresponding crypto graph to be shown or hidden. There is a note at the bottom of the slides containing charts which indicate the available triggers.
